<Admonition type="warning" title="Deprecation notice: Azure regions on Neon">
**Starting April 2, 2026:** You can no longer **create new Neon projects** in **Azure** regions. Neon **Azure** regions (`azure-eastus2`, `azure-westus3`, `azure-gwc`) are **deprecated**. **Migration deadlines** for existing projects will be communicated by **email from Neon** and in the **[Neon changelog](/docs/changelog)**.

If your database runs on an Azure region, please start planning for a **data migration**. See **[Region migration](/docs/import/region-migration)** for supported migration paths.
</Admonition>

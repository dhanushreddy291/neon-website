## Postgres extension updates

### neon extension version update

The **`neon`** extension is now **version 1.14**, up from **1.9**. The `neon` extension can be used to monitor **[Local File Cache](/docs/extensions/neon#what-is-the-local-file-cache)** usage (how often pages are served from cache versus Neon storage) through the **`neon_stat_file_cache`** view and **`EXPLAIN ANALYZE`**. See **[The neon extension](/docs/extensions/neon)** for details.

### pg_search deprecation

Neon support for the **`pg_search`** extension is deprecated. As of **March 19, 2026**, it is not available for **new** Neon projects. If you already use it, please start planning a migration. See **[The pg_search extension](/docs/extensions/pg_search)** for alternatives.

## Azure region deprecation

As of **April 2, 2026**, Neon no longer supports creating **new** projects in Azure regions, and support for `azure-eastus2`, `azure-westus3`, and `azure-gwc` regions is deprecated.

If your Neon project currently runs in an Azure region, we will email you about timelines for migrating. Please see **[Region migration](/docs/import/region-migration)** for region migration options.

<Admonition type="tip" title="How to check if you are on an Azure region">
Open your organization’s **[Projects](https://console.neon.tech/app/)** page. Your project’s region is shown in the **Projects** table.
</Admonition>

<details>
<summary>**Neon API**</summary>

The **`GET /consumption_history/account`** endpoint ([Get account consumption metrics](https://api-docs.neon.tech/reference/getconsumptionhistoryperaccount)) is **deprecated**, with a planned sunset of **June 1, 2026**. If you use aggregated account-level consumption metrics, move to **[project-level consumption metrics](/docs/guides/consumption-metrics)** before that date. The [project-level endpoint](https://api-docs.neon.tech/reference/getconsumptionhistoryperprojectv2) reports metrics Neon's usage-based billing metrics for improved consumption tracking.
